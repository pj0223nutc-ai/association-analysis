Step 1 — 認識資料

這步是要瞭解資料的基本結構。

在 Excel 中怎麼做

  • 開啟出庫資料 Excel 檔
  • 觀察欄位：訂單編號、商品名稱
  • 用「資料→移除重複項」計算不重複訂單數
  • 同法計算不重複商品數

常見錯誤

  • 把記錄筆數當作訂單數（一筆訂單有多行）
  • 忘記先選對欄位再移除重複

Step 2 — 篩選目標商品

找出包含「黃金搭檔（兒童青少年）」的訂單。

在 Excel 中怎麼做

  • 在「商品名稱」欄位加上篩選
  • 篩選出 = 黃金搭檔（兒童青少年）
  • 複製出現的「訂單編號」欄到新工作表
  • 移除重複，計算個數

常見錯誤

  • 沒有去重複，直接數行數

Step 3 — 統計共現頻次

在目標商品訂單裡，其他商品出現幾次（訂單筆數，非數量）。

在 Excel 中怎麼做

  • 用前步驟的 34 個訂單編號
  • 回原資料以 VLOOKUP 或篩選方式找出這 34 筆訂單的全部明細
  • 建立商品樞紐分析表，以「訂單編號」計算不重複個數

注意

  • 計算「訂單筆數」不是「購買數量」
  • 目標商品本身也要排除在外

Step 4 — 計算支援度

Support = 兩商品同時出現的訂單數 ÷ 全部訂單數。

公式

Support = |X∩Y| / N

  • |X∩Y| = 兩商品共同出現次數
  • N = 全部訂單數 = 984

Excel 操作

  • 在儲存格輸入 =共現次數/984
  • 格式設為「數字，小數 4 位」

常見錯誤

  • 分母用 34（目標訂單數）而非 984（全部）

Step 5 — 計算信賴度

Confidence = 兩商品共現訂單數 ÷ 目標商品訂單數。

公式

Confidence = |X∩Y| / |X|

  • |X∩Y| = 共現次數
  • |X| = 目標商品訂單數 = 34

Excel 操作

  • 在儲存格輸入 =共現次數/34

常見錯誤

  • 把支援度和信賴度的分母搞混

Step 6 — 判斷關聯商品

同時滿足最小支援度 AND 最小信賴度才算關聯性商品。

判斷條件

  • 最小支援度 ≥ 0.03
  • 最小信賴度 ≥ 0.40
  • 兩個條件都要滿足！

關聯規則的意義

  • 支援度：整體上有多常一起買
  • 信賴度：買了 X 之後有多大機率也買 Y

計算支援度 (Support)

兩商品同時出現佔全部訂單的比例
🔒
▼
學習目標
支援度（Support）衡量兩種商品「同時出現」在整體訂單中的頻率。 分母是全部訂單數 984（不是目標訂單數 34）。 最小支援度門檻 = 0.03，低於此值的商品對就不值得進一步分析。
支援度公式
Support(X→Y) = |X∩Y| / N
範例：Support(兒童青少年 → 女士)
= 33 / 984 = 0.0335

計算信賴度 (Confidence)

買了目標商品的人，有多少比例也買了另一種？
🔒
▼
學習目標
信賴度（Confidence）衡量「在購買 X 的訂單中，有多少比例同時購買了 Y」。 分母是目標商品訂單數 34（不是全部訂單 984）。 最小信賴度門檻 = 0.4，代表至少 40% 的目標訂單也有購買該商品。
信賴度公式
Confidence(X→Y) = |X∩Y| / |X|
範例：Confidence(兒童青少年 → 女士)
= 33 / 34 = 0.9706

最終結論

在本次分析中，目標商品「黃金搭檔（兒童青少年）」在 984 筆訂單中出現於 34 筆（3.46%）。 在這 34 筆訂單中，只有「黃金搭檔（女士）」同時滿足最小支援度（0.0335 ≥ 0.03） 與最小信賴度（0.9706 ≥ 0.40）兩項門檻。

其他三種商品（威士雅金絲燕窩、張裕葡萄酒禮盒、劍南春52度）雖然在某些訂單中共現， 但或信賴度夠高而支援度不足，或兩者均未達標，因此不構成關聯性商品。

商業應用建議
  • 貨架擺放：將「黃金搭檔（兒童青少年）」和「黃金搭檔（女士）」相鄰陳列， 方便顧客一次找到兩件。
  • 促銷組合：設計「兒童青少年 + 女士」的家庭套裝組合，享有優惠價格。
  • 精準推薦：當顧客加入「兒童青少年」至購物車時，自動推薦「女士」版本。
  • 庫存管理：兩款商品的進貨量應同步規劃，避免其中一款缺貨影響組合銷售。
