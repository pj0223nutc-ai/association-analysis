984 筆訂單 3,333 筆出庫記錄 Apriori 關聯規則
步驟說明 & Excel 操作對照

步驟 1：認識資料

這份資料來自一個配送中心的出庫記錄，格式為「一筆訂單 → 多個商品」。

Excel 操作對照

  • 在 Excel 中，原始資料通常為兩欄：訂單編號 / 商品名稱
  • 用「移除重複」功能統計不重複訂單數
  • 用「COUNTIF + 樞紐分析」統計商品種類數

小技巧

注意：一筆訂單可能包含多個商品，所以原始記錄數（3,333）遠大於訂單數（984）。

步驟 2：篩選目標商品訂單

找出所有「包含目標商品 X」的訂單，這是計算支援度的分子第一步。

Excel 操作

  • 篩選欄位：在商品名稱欄位使用「篩選」功能
  • 選取目標商品後，複製訂單編號那欄
  • 貼到新工作表，再用「移除重複」統計不重複訂單數

為什麼要去重複？

如果一筆訂單裡有兩件目標商品，直接計數會算兩次，所以要先去重複再算。

步驟 3：統計共現商品頻次

在 34 筆目標訂單中，計算每種商品「出現在幾筆不同訂單」。

Excel 操作（樞紐分析表）

  • 先篩選出 34 筆目標訂單的所有記錄（共 119 筆）
  • 插入 → 樞紐分析表
  • 將「商品名稱」拖到「列」，將「訂單編號」拖到「值」（計數）
  • 排序：從大到小

注意

目標商品本身（黃金搭檔兒童青少年）也會出現在統計結果中（34次），這是正常的。

步驟 4：計算支援度

Support(X→Y) = |X∩Y| / N

|X∩Y| = 兩個商品同時出現在幾筆訂單
N = 全部訂單總數（984）

Excel 操作

  • 在步驟 3 的頻次表旁邊新增一欄「支援度」
  • 公式：=共現次數/984
  • 設定門檻：最小支援度 = 0.03（即 3%）

解讀

支援度 0.0335 代表有 3.35% 的訂單同時包含這兩件商品。

步驟 5：計算信賴度

Confidence(X→Y) = |X∩Y| / |X|

|X| = 包含 X（目標商品）的訂單數（34）

Excel 操作

  • 在支援度旁邊新增一欄「信賴度」
  • 公式：=共現次數/34
  • 設定門檻：最小信賴度 = 0.4（即 40%）

解讀

信賴度 0.9706 代表：買了 X 的訂單中，有 97.06% 也買了 Y。

步驟 6：判斷關聯商品

必須同時滿足兩個條件：
✅ 支援度 ≥ 0.03
✅ 信賴度 ≥ 0.4

Excel 操作

  • 新增「是否為關聯商品」欄位
  • 公式：=IF(AND(支援度>=0.03, 信賴度>=0.4), "是", "否")

業務意義

找到關聯商品後，可以在推薦系統中設定：「買了 X 的客戶，推薦商品 Y」。這就是購物車分析的核心！

計算支援度

🔒
▼
本步驟目標
計算每個候選商品對目標商品的支援度。支援度代表「兩個商品同時出現」在所有訂單中的比例。
公式
Support(X→Y) = |X∩Y| ÷ N
|X∩Y| = 兩商品共同出現的訂單數  |  N = 總訂單數（984）

計算信賴度

Confidence(X→Y) = |X∩Y| / |X|
🔒
▼
本步驟目標
計算信賴度：在「含目標商品 X」的訂單中，有多少比例「也含 Y」？ 信賴度越高，代表關聯性越強。
公式
Confidence(X→Y) = |X∩Y| ÷ |X|
|X| = 包含目標商品 X 的訂單數（34）
🎯 信賴度計算機

填入共現次數，點擊「計算」自動算出信賴度。最小信賴度門檻 = 0.4

✏️ 輸入信賴度（小數點後 4 位，誤差 ±0.001 可接受）
