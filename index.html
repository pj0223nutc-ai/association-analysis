什麼是關聯規則分析？

Association Rule Mining

關聯規則分析是一種資料挖掘技術，用於發現大量交易資料中項目之間的有趣關係。最典型的應用場景是「購物籃分析」（Market Basket Analysis）：透過分析顧客購買記錄，找出哪些商品常常被一起購買，進而輔助商品陳列、促銷策略與推薦系統設計。

例如，若超市發現「購買啤酒的顧客中，有 68% 也會購買尿布」，這條規則就可以指導商品擺放或捆綁促銷決策。

🍺

啤酒與尿布的故事

The Classic Example

1990 年代，美國 Walmart 在分析購物資料時發現：每逢週五傍晚，啤酒與嬰兒尿布常常一起被購買。推測原因是爸爸在下班後順路買尿布，同時順手拿了啤酒。

這個發現讓 Walmart 將啤酒與尿布擺放在相鄰貨架，大幅提升了兩者的銷售量。這正是關聯分析在實務中的價值。

📐

三大核心指標公式

Support · Confidence · Lift

-- 設 X = 目標商品, Y = 候選商品, N = 總訂單數 -- Support(X → Y) = |X ∩ Y| / N -- X 與 Y 同時出現的訂單數 / 總訂單數 Confidence(X → Y) = |X ∩ Y| / |X| -- 買了 X 的訂單中，也買了 Y 的比例 Lift(X → Y) = Confidence / (Support(Y)) = (|X ∩ Y| / |X|) / (|Y| / N) -- Lift > 1: 正向關聯 | = 1: 獨立 | < 1: 負向
🧮

計算範例

Worked Example

假設有 100 筆訂單，其中：

條件數量
含「咖啡」的訂單 |X|40 筆
含「牛奶」的訂單 |Y|30 筆
同時含咖啡＋牛奶 |X∩Y|20 筆
總訂單 N100 筆
Support(咖啡→牛奶) = 20 / 100 = 0.20 (20%) Confidence(咖啡→牛奶) = 20 / 40 = 0.50 (50%) Lift(咖啡→牛奶) = 0.50 / (30/100) = 1.67

Lift = 1.67 > 1，表示購買咖啡會「提升」購買牛奶的機率 67%，具有顯著正向關聯。

如何解讀分析結果？

Result Interpretation Guide

指標數值範圍意義建議閾值
Support0 – 1規律的普遍程度；過低表示樣本稀少≥ 0.03（3%）
Confidence0 – 1規律的可靠程度；越高購買概率越確定≥ 0.40（40%）
Lift0 – ∞>1: 正向 / =1: 無關 / <1: 負向≥ 1.0
⚠️ 常見陷阱
  • 高信賴度不代表高 Lift：若 Y 本身出現率很高（如「塑膠袋」），任何商品與之的 Confidence 都高，但 Lift 可能接近 1
  • 支援度過低的規律可能只是偶然巧合，建議設定最小支援度過濾
  • 關聯規則只反映相關性，不代表因果關係
